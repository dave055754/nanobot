# ledger-validation skill 校验结果字段不一致修复计划

## 问题描述
ledger-validation skill 中校验结果返回存在以下问题：
1. `validateflag` 字段与实际校验结果不一致
2. 校验失败时返回的 `valid` 字段仍然为 `true`，应该为 `false`

## 问题分析

### 当前代码逻辑
在 `nanobot/skills/ledger-validation/scripts/validate_ledger.py` 中：

```python
# 第96-100行：构建返回结果
result = {
    "valid": len(errors) == 0,  # 问题：这个逻辑可能存在bug
    "errors": errors,
    "data": data
}
```

### 问题点
1. **字段命名不一致**：用户提到 `validateflag` 字段，但代码中使用的是 `valid` 字段
2. **逻辑错误**：当存在校验错误时，`valid` 字段应该返回 `false`，但实际可能返回 `true`
3. **错误信息不完整**：返回的错误信息可能没有准确反映实际的校验失败原因
